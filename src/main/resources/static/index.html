chatApi Documentation

Open-source realtime messaging and matchmaking platform built with Spring Boot, WebSocket/STOMP, WebRTC signaling, and MongoDB.

v4.0.0 Spring Boot 3.4.5 Java 17 MongoDB MIT License

API Documentation

Chat Endpoints

Method Endpoint Description Auth
POST /api/chat Send message Token
GET /api/chat/{id} Get message Token
GET /api/chat/history Chat history Token
PUT /api/chat/{id} Edit message Token
DELETE /api/chat/{id} Delete message Token

Queue Endpoints

Method Endpoint Description Auth
POST /api/queue Join queue Token
GET /api/queue/{sessionId} Queue status Token
GET /api/queue View all Admin
PUT /api/queue/{sessionId} Update entry Token
DELETE /api/queue/{sessionId} Leave queue Token

Session Endpoints

Method Endpoint Description Auth
POST /api/sessions Start session Token
GET /api/sessions/{id} Session info Token
GET /api/sessions List all Admin
PUT /api/sessions/{id} Modify session Token
DELETE /api/sessions/{id} End session Token

STOMP Endpoints

Destination Description Payload
/session.message Chat message ChatMessage
/session.signal WebRTC signaling SignalingMessage
/session.disconnect Leave session String
/queue.join Enter matchmaking String
/queue.leave Exit matchmaking String

Contributing

Thanks for your interest in contributing! chatApi is an open-source project and welcomes issues, bug reports, and pull requests. Please read the project on GitHub and the README.md for setup instructions, coding guidelines, and the contribution workflow.

Quick Start

  1. Fork the repository and create a feature branch.
  2. Run the app locally with Java 17 and Maven:
mvn spring-boot:run

If you're adding features, include tests where appropriate and open a pull request with a clear description of the change. For questions, open an issue on GitHub.
